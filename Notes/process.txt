Release Proceedure:
- Merge all changes into master
- Close VS Code and Factorio, run build_release-script
- Upload new release zip-file to the mod portal
- For each new feature branch, run new_changelog_entry-script


MISC:
- Count LOC: ( find ./ -name '*.lua' -print0 | xargs -0 cat ) | wc -l
- Delete most recent local commit: git reset --hard HEAD^
- Force-replace commit on origin/: git push --force origin
- Merge develop to master: git checkout master; git merge develop
- Add local branch to remote repo: git push -u origin develop
- Remove local and remote branch: git branch -d develop; git push origin --delete develop
